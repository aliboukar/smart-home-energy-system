# Morphology of Smart Home Energy Management System

## 1. Information Structure
- **Data used:** 
  - Energy consumption from smart devices (lights, appliances, thermostat)  
  - Sensor readings (temperature, motion, light)  
  - User preferences and schedules  
  - Weather and external data for optimization

## 2. Technical Structure
- **Where it runs:** 
  - Home IoT devices and sensors  
  - Local hub (e.g., Raspberry Pi)  
  - Cloud servers for data processing and analytics  
  - Mobile and web apps for user control

## 3. Organisational Structure
- **Who interacts:** 
  - Homeowners (users controlling devices)  
  - Energy providers (optional integration)  
  - System administrators and developers (maintenance and updates)

## 4. Optional Technological / Spatial Layers
- Smartphone and tablet apps  
- Voice assistants (Alexa, Google Home)  
- Local home network and cloud backend
